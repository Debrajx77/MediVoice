# MediVoice

MediVoice is a single-page symptom translator that turns plain-language symptom descriptions into calm triage guidance. It explains what symptoms could indicate, how urgent they may be, one useful next action, and what to bring to a doctor.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Add your Gemini API key:
   ```bash
   cp .env.example .env
   ```
   Then edit `.env` and set `VITE_GEMINI_API_KEY`.

3. Start the app:
   ```bash
   npm run dev
   ```

## Responsible Use

This tool does not replace professional medical advice. If symptoms feel severe, sudden, or frightening, seek urgent medical care immediately.
